Model equations

\[ \frac{\partial a}{\partial t} = D_A \nabla^2 a - ab^2 + r(1-a) \\ \frac{\partial b}{\partial t} = D_B \nabla^2 b + ab^2 - k_1 b - k_2 bc^2 \\ \frac{\partial c}{\partial t} = D_C \nabla^2 c + k_2 bc^2 - k_3 c \\ \]

Model by
